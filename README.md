# CurateSage - Personalized Course Recommendation System

CurateSage is an innovative graph neural network solution that leverages cutting-edge machine learning techniques to make course recommendations. At its core, the system employs a sophisticated recommendation engine powered by Heterogeneous Graph Neural Networks (GNN). These advanced algorithms analyze a student's academic history, interests in the form of text-based inputs, and career goals to generate highly personalized course suggestions.
